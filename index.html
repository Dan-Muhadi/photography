How to create a photography gallary

Using this module you can create a collection of images that are located in a common folder. When inserting the photo gallery module into a page the system will basically insert resized thumbnails of your images and apply a light box effect. The thumbnails are sorted by file name alphabetically. Using the photo gallery module consists of the following steps

Follow simple steps

  • Create the folder the gallery's images will be stored into and upload the images
  • Create a photo gallery module and select the folder created at step 1
  • Insert the module in a page
